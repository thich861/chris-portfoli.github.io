VIRCL — Hypervisor & VM Backup

Specialisation project · Proxmox & VMware ESXi

Proxmox VMware ESXi Backup & Recovery Virtualization Snapshots

Overview

VIRCL was a specialisation project focused on backup strategies for hypervisors and virtual machines. The project covered both Proxmox VE and VMware ESXi environments, exploring how to protect VM workloads through scheduled backups, snapshots, and offsite replication.

What I did

I configured backup jobs on Proxmox using PBS (Proxmox Backup Server) and explored native snapshot capabilities on VMware ESXi. The goal was to ensure minimal data loss and fast recovery in the event of a hypervisor failure or VM corruption.
